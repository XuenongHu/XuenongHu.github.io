# Xuenong Hu's Personal Portfolio Website

Welcome to the GitHub repository for my personal website! This site showcases my academic background, research engineering experience, projects, and publications.

## 🌐 Live Website
You can view the live site here: [xuenonghu.github.io](https://xuenonghu.github.io)

## 📁 Repository Structure

*   `index.html` - The homepage of the website.
*   `about.html` - Information about my background and journey.
*   `experience.html` - Details regarding my professional history as a Research Engineer.
*   `projects.html` & `publications.html` - Showcases of my engineering projects and academic papers (including work on transient stability).
*   `blog.html` & `/blog` - Personal insights, articles, and updates.
*   `style.css` & `script.js` - The styling and interactive functionality of the site.
