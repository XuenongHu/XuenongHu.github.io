Engineering Projects

A selective breakdown of computational models, optimization algorithms, and infrastructure systems built to handle multi-tier energy transmission challenges.

Transient Stability & Frequency Response Simulator

Developed an analytical simulation framework to model multi-bus power grid frequency response following high-impact generator contingencies. Integrated synthetic Battery Energy Storage System (BESS) secondary frequency control loops to successfully mitigate under-frequency load shedding (UFLS) dynamics.

  • MATLAB
  • Simulink
  • Transient Analysis

BESS Dispatch Optimizer for Curtailment Mitigation

Programmed a predictive optimization engine using mixed-integer linear programming (MILP) to schedule daily charging/discharging profiles for co-located wind/solar storage assets. Maximized locational marginal pricing (LMP) arbitrage while minimizing grid capacity bottlenecks.

  • Python
  • NumPy / SciPy
  • Grid Management

Low-Carbon Thermal Plant Retrofit Calculator

Engineered an object-oriented thermodynamic simulation tool to evaluate retrofitting legacy coal and gas turbine infrastructure. Models efficiency deltas, biomass co-firing ratios, and carbon capture carbon absorption loops to align utility operations with strict emission compliance standards.

  • Thermodynamics
  • Data Modeling
  • Python

Virtual Power Plant (VPP) Aggregator Dashboard

Designed an architectural system dashboard aggregating multi-tier distributed energy resources (commercial solar arrays, industrial batteries, EV fast-chargers) into a unified grid-responsive network. Features low-latency mock telemetry handling simulated automated demand-response events.

  • Smart Grids
